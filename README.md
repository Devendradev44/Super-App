# Super App

## Features

- User Registration
- Category Selection
- Personalized Dashboard
- Weather Widget
- News Widget
- Notes Widget
- Countdown Timer
- Movie Recommendations

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Zustand
- Axios
- React Hook Form
- Zod

## Installation

npm install
npm run dev

## Build

npm run build

## Environment Variables

NEXT_PUBLIC_OPENWEATHER_API_KEY=
NEXT_PUBLIC_NEWS_API_KEY=
NEXT_PUBLIC_OMDB_API_KEY=
